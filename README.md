# malyavka -- проект для обучения мелких особей
Выложено тут http://malyavka.dweiner.ru/ 

## Технологии
* Node.js
* Vue.js

## Установка локально
```
    npm install
    npm run serve
```

## Установка на сервере
```
npm run build
```

## Пережатие картинок
```
 mogrify -geometry 'x600>' -quality 95 *
```
